# Sistema de Gestión de Inventario Web

Proyecto desarrollado en PHP nativo y MySQL para la gestión básica de inventario, incluyendo autenticación de usuarios, control de productos y registro de movimientos.



##  Tecnologías utilizadas
- PHP (nativo)
- MySQL
- HTML
- Bootstrap 5
- JavaScript (básico)


##  Funcionalidades

###  Autenticación
- Login de usuarios con sesiones en PHP
- Protección de rutas internas

###  Gestión de productos
- Crear productos
- Listar productos
- Eliminar productos
- Control de stock

###  Control de inventario
- Entrada de productos (+)
- Salida de productos (-)
- Validación para evitar stock negativo

###  Dashboard
- Total de productos
- Productos con bajo stock
- Últimos movimientos registrados

###  Historial
- Registro de entradas y salidas con fecha

##  Base de datos
Tablas principales:
- usuarios
- productos
- movimientos


##  Objetivo del proyecto
Simular un sistema real de inventario para fortalecer habilidades en desarrollo backend con PHP y manejo de bases de datos MySQL.


##  Autor
Desarrollado como proyecto de práctica para fortalecimiento de habilidades en desarrollo web backend.
